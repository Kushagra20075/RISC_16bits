# RISC_16bits
A simple 16 bit RISC Assembler and Simulator
Supports the functions
Addition
Subtraction
Move Immediate
Move Register
Load Store Multiply
Divide
Right Shift
Left Shift
XOR
Or
And
Invert
Compare
Unconditional Jump
Jump if less than
Jump if equal to
Jump if greater than
Jump if equal
Halt
Handles Flags Register for Overflow
